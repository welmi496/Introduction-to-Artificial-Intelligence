# Clustering and Dimensionality Reduction Analysis on the Wine Dataset

## Project Overview
This project applies clustering and dimensionality reduction techniques to the Wine dataset using Python in Google Colab. The objective is to identify natural groupings in the data and visualize high-dimensional patterns in a lower-dimensional space.

## Dataset
The dataset used is the Wine dataset from `sklearn.datasets`.

## Methods Used
- Data preprocessing
- Feature scaling with StandardScaler
- K-Means Clustering
- Hierarchical Clustering
- Elbow Method
- Silhouette Score
- Principal Component Analysis (PCA)
- t-SNE visualization

## Files
- `Clustering_Assignment.ipynb` - Google Colab notebook containing all code and analysis
- `README.md` - project overview and instructions
- `report.pdf` - two-page report summarizing findings

## How to Run
1. Open the notebook in Google Colab.
2. Run all cells in order.
3. Review the clustering results, silhouette scores, and visualizations.
4. Export the notebook or report as PDF for submission.

## Output
The notebook produces:
- Elbow method plot
- Silhouette score comparison
- Dendrogram
- PCA cluster plots
- t-SNE visualization
- Model performance comparison table
